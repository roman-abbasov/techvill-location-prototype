# Тестовое задание 1 Техвилл

Интерактивный демонстрационный прототип для оценки потенциала новых точек розничной сети на основе геоаналитики. Все бизнес-данные синтетические.

## Возможности

- презентационная вкладка с решением PM-задачи;
- Яндекс.Карта с Heatmap, существующими магазинами и рекомендациями;
- фильтры радиуса и максимальной аренды;
- рейтинг локаций с прогнозом выручки, окупаемостью и каннибализацией;
- подробная карточка с вкладом факторов и объяснением;
- демонстрационная LLM-кнопка без сетевого вызова.

## Локальный запуск

Требуется Node.js 20 или новее.

```powershell
npm install
Copy-Item .env.example .env
# вставить ключ в VITE_YANDEX_MAPS_KEY внутри .env
npm run generate:data
npm run dev
```

Откройте адрес, который Vite напечатает в терминале. Ключ Яндекс.Карт можно получить на [странице разработчика](https://yandex.ru/dev/maps/). Если ключ не указан, приложение покажет инструкцию вместо карты, а рейтинг и карточки продолжат работать.

## Проверка

```powershell
npm test -- --run
npm run build
```

## Переменные окружения

```env
VITE_YANDEX_MAPS_KEY=
```

`.env` исключён из Git. Не добавляйте ключ в исходный код или `.env.example`.

## Ограничения демо

Скоринг и риск каннибализации заранее рассчитываются offline-скриптом. Кнопка «Сформировать ИИ-объяснение» намеренно показывает демонстрационное уведомление: backend и живой LLM-вызов не входят в текущий прототип.
